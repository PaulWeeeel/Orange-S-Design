# Orange-S-Design
A class quest
操作系统课程设计-简单操作系统  
重新实现一个系统模块+用户级应用程序

在Ubuntu系统下Ctrl+Alt+T呼出终端，在项目文件下输入bochs即开始运行系统，默认有断点，再输入c继续运行 
在XOS下输入命令运行以下程序:
1.calculate 你的算式 ....运行计算器计算结果
2.figlet 任意键盘输入 ....键盘输入转化为字符画
3.puzzle            ....拼图游戏
4.calendar 年份 0/1 ...显示万年历的上（0）下（1）半年
5.proc              ...显示进程状态表
6.ls                ....显示文件列表
